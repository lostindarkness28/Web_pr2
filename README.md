#  Система моніторингу підстанції 110/10 кВ

Інтерактивний дашборд для дистанційного контролю параметрів силового трансформатора та фідерів у режимі реального часу. Проєкт реалізовано як односторінковий веб-додаток (SPA) з високим рівнем візуалізації даних.

## Технології та інструменти
* **HTML5 / CSS3** — інтерфейс із підтримкою світлої та темної тем.
* **JavaScript**  — обробка логіки, симуляція датчиків та керування станами.
* **Chart.js** — бібліотека для динамічної побудови лінійних та кругових графіків.
* **Web Audio API** — звукова сигналізація при досягненні критичних значень.

## Структура проекту
1. Переконайтеся, що у вас є наступні файли та папки в одній директорії проєкту:
   * `index.html` — головний файл;
   * `css/style.css` — стилі;
   * `js/script.js` — логіка;
   * `images/shema.jpg` — фонова схема.
2. Відкрийте файл **index.html** через браузер або за допомогою **Live Server** у VS Code.

## Запуск та використання
1. Відкрийте файл `index.html` у будь-якому сучасному браузері.
2. Для активації автоматичного режиму натисніть кнопку **"Авто-оновлення"**.
3. При виникненні аварійних значень (червона індикація) система автоматично подасть звуковий сигнал.
4. Історію вимірювань можна завантажити у форматі **CSV** для подальшого аналізу.
